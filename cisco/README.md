# cml2_sample

## prerequisites

+ windows PC
  + CML2 (as server)
  + ngrok
+ mac or linux
  + this repo
    + virl2_client
    + pyats
      + genie
      + testbed
      + ...


```sh
make all
# or make rtest for the 2nd time
make test
```
